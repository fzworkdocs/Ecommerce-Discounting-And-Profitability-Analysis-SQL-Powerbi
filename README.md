# E-Commerce Discounting And Profitability Analysis
### PostgreSQL + Power BI Project

## Project Overview
This project analyzes an e-commerce dataset using PostgreSQL and Power BI to evaluate sales performance, profitability, customer behavior, delivery operations, and discounting strategies.

The analysis focuses on identifying the impact of discounts on profit margins and discovering key business insights through interactive dashboards and SQL-based analysis.

## Tools & Technologies
- PostgreSQL
- Power BI
- pgadmin
- DAX
- Power Query

## Key Insights
- Medium discount ranges (20–35%) generated the highest profitability.
- Fashion category was the top contributor to both sales and profit.
- Electronics category underperformed due to lower discount competitiveness.
- Deliveries completed within 4–7 days produced the best profit margins.
- Mid-Tier and VIP customers contributed the highest overall profits.

## Recommendations
- Improve discount competitiveness in Electronics products.
- Increase marketing investment in Fashion products.
- Optimize delivery operations to reduce slow shipments.
- Implement loyalty programs to convert Mid-Tier customers into VIP customers.
